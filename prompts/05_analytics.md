# ⑤ 分析AI:週次レビュー

data/ 内の楽天アフィリエイトレポート(CSV)を読み込み分析する。

入力: クリック数/CV数/報酬額/投稿ジャンル別データ(ROOM経由/直リンクの内訳があれば含む)
出力:
1. 勝ちジャンルTOP3と負けジャンル
2. クリックはあるがCVしない商品の原因仮説
3. 来週の投稿配分の修正案(→ ④に反映)
4. Bランク条件(#オリジナル写真の月間枚数)の進捗確認リマインド
5. ROOM経由 vs 直リンクのCV率比較(直リンクを併用している場合のみ)。CV率差が明確なら、CLAUDE.mdの「導線の使い分けルール」に基づきフェーズ切替(ROOM集約→直リンク併用)を提案する
6. CLAUDE.mdの「現在の運用フェーズ」セクションを更新すべきか明記(据え置き/切替提案のどちらか)

初月はクリック数のみをKPIとする(報酬確定まで約2ヶ月ラグがあるため)。フェーズ切替はAランク到達(月1,500〜2,000クリック圏)が前提条件。
