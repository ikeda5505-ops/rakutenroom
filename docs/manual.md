# 楽天ROOM 完全スタートマニュアル(要点)

初めての登録・準備は docs/getting_started.md に詳細手順あり(楽天アフィリエイトへのサイト登録など見落としやすい必須ステップを含む)。このファイルは要点のみ。

## 開設(15分)
1. 楽天IDを用意 → ROOMアプリをインストール → ログイン
2. ユーザー名・アイコン・プロフィール設定(docs/profile.md 参照)
3. コレクション7つ作成(prompts/03_room_design.md 参照)
4. **楽天アフィリエイトの「サイト情報」にROOMのURLを登録**(未登録は規約違反扱い・ボーナス対象外・垢BANリスク。詳細はdocs/getting_started.md手順4)

## 投稿の基本
- 検索 → 商品ページ →「コレ!」→ 投稿文貼付 → 投稿
- 自分で撮影した写真には #オリジナル写真 タグ必須(ランクの生命線)

## ランク制度
- E→D→C→B→A→S。毎月末判定・翌月1日更新
- B(+1%): C条件+#オリジナル写真を月の指定枚数(アプリで確認)
- A(+2%): 誘導数上位。目安 月1,500〜2,000クリック
- S(+3%): さらに上位。人数枠制
- ボーナスは1商品上限1,000円
- 検索/フォロー/いいねを3ヶ月放置でC以下に降格

## 初月ロードマップ
- Day1: 開設+C条件消化+初投稿10件(棚を埋めてからフォロー回り)
- Day2-7: 1日3投稿+#オリジナル写真週2-3枚+毎日いいね10-20件
- Day8-30: B条件達成、X連携シェア週2-3回、月末に⑤で分析

## 禁止事項
自動化ツール全般 / 説明文コピペ / 体験談捏造 / 楽天市場以外の商品

## 報酬
楽天アフィリエイト管理画面で確認。発生→支払いまで約2ヶ月。
通常報酬2〜4%(一部最大20%)+ランクボーナス。
