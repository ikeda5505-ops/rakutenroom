# はじめての準備・登録手順(完全版)

Web検索で最新の登録フローを確認した上でまとめた、ゼロから始めるためのチェックリスト。上から順番にやればOK。
所要時間目安: 全体で30〜45分(コレクション作成・初投稿10件を除く)。

━━━━━━━━━━━━━━━━━━━━
## 0. 事前に準備するもの
━━━━━━━━━━━━━━━━━━━━
- [ ] 楽天会員ID・パスワード(未登録なら楽天市場トップから「新規会員登録」)
- [ ] **スマホ(ROOMアプリ)は必須**。PCブラウザ版のROOMは投稿・コレクション作成などの主要機能が使えない(閲覧専用に近い)ため、実作業は基本アプリで行う
- [ ] PC(楽天アフィリエイト管理画面の設定用。こちらはPCブラウザで問題ない)
- [ ] プロフィール画像・ヘッダー画像: `assets/room_icon.png` / `assets/room_header.png`(準備済み)
- [ ] 報酬受け取り用の楽天カード または 楽天銀行口座(後述、今のうちに用意しておくと後で困らない)

━━━━━━━━━━━━━━━━━━━━
## 1. 楽天ROOMに登録する(スマホアプリ)
━━━━━━━━━━━━━━━━━━━━
1. スマホに**楽天ROOMアプリ**をインストール(App Store/Google Play)
2. 「アカウント登録/ログイン」→ 楽天ID・パスワードでログイン(**これだけでROOM登録は完了**。別途の会員登録フォームはない)
3. 初期設定画面が出る:
   - 「次へ」を3回タップ
   - 「興味があるジャンル」を3つ以上選択(暮らし・キッチン・美容など、このROOMのコンセプトに近いものを選ぶ)
   - 「選択したジャンルのおすすめユーザーをフォロー」に**必ずチェック**(50〜100人を自動フォローでき、初期の相互訪問・いいね回りの母数になる)

━━━━━━━━━━━━━━━━━━━━
## 2. プロフィール(マイページ)を設定する(アプリ)
━━━━━━━━━━━━━━━━━━━━
ROOM内の「マイページ」→「プロフィール編集」から設定。docs/profile.mdの確定版をそのまま使う。

- [ ] **ユーザー名**: `kurashi_lab_room` を第一候補に(空いていなければ `raku_kaiteki` / `smart_kurashi`)
- [ ] **表示名**: 「みつ|暮らしを快適にするモノ選び」
- [ ] **プロフィール画像**: `assets/room_icon.png` をアップロード(推奨:1000×1000px以上の正方形。準備済み画像はこの要件を満たす想定)
- [ ] **ヘッダー画像**: `assets/room_header.png`
- [ ] **プロフィール文**: docs/profile.mdの確定版をそのままコピペ
  ```
  🏠 「これは本当に良かった」暮らしの名品だけを厳選しています
  ✅ 選定基準:レビュー100件以上・★4.3以上をクリアした商品のみ
  ☀️ 今の特集 → 猛暑サバイバル決定版 & お中元ラストスパート
  📝 「結局どっちがいいの?」を口コミ分析であらかじめ悩んでおきます
  🕘 更新は毎日、朝7時・昼12時・夜21時の3回
  💬 いいね・フォローから来てくださった方、お返し伺います✨
  ```
- [ ] 設定後、**自分のROOMのURL**(`room.rakuten.co.jp/ユーザー名` の形)をメモしておく → 手順3で使う

━━━━━━━━━━━━━━━━━━━━
## 3. 【最重要・見落とし注意】楽天アフィリエイトへの「サイト登録」(PC可)
━━━━━━━━━━━━━━━━━━━━
**ここが今回いちばん重要なポイントです。** 楽天ROOMでの活動は自動的に楽天アフィリエイトの仕組みに乗っかっていますが、「サイト情報」として**ROOMのURLを別途登録しないと規約違反扱い**になり、キャンペーン・ボーナス対象外や、最悪アカウント停止(垢BAN)・料率BANのリスクがあると複数の情報源で報告されています。ROOM登録が終わったら**必ず**このステップをやってください。

1. [楽天アフィリエイト・サイト情報登録ページ](https://affiliate.rakuten.co.jp/user/sites)にアクセス(楽天IDでログイン。新規の「会員登録」画面が出たら利用規約に同意して進めばOK。これで自動的にアフィリエイト会員としてのマイページが使えるようになる)
2. 「サイト情報の登録・変更」で以下4項目を入力:
   - サイト名(例:くらし研究室|暮らしを快適にするモノ選び)
   - サイトURL(手順2でメモした**ROOMのマイページURL**)
   - サイトのジャンル(近いものでOK、選択が実際の紹介商品を縛るものではない)
   - サイトで取り扱う商品ジャンル
   - 順位: サイトがROOMのみなら「1位」でOK
3. 保存して登録完了。今後ブログ(WordPress)を追加する際も同じ画面で追加登録し、複数サイトになったら「順位登録」(成果報酬・閲覧数が多い順)も行う

**楽天アフィリエイトの「マイページ」でできること**(こちらも今のうちに一度目を通しておくと良い):
- アフィリエイトリンクの生成(ブログ用の直リンクはここで作る)
- 成果レポートの閲覧(⑤分析AIの入力データはここからCSVダウンロード)
- バナー素材のダウンロード
- 各種設定(サイト情報・報酬受け取り設定など)

━━━━━━━━━━━━━━━━━━━━
## 4. 報酬の受け取り設定(後回し可・PC可)
━━━━━━━━━━━━━━━━━━━━
- 通常報酬が**月間3,000円を超えると**、報酬受け取りに楽天カード または 楽天銀行のアカウントID連携が必須になる
- **今すぐ必須ではない**。最初は楽天キャッシュでの受け取りが一般的。余裕があるうちに楽天カード/楽天銀行の連携だけ済ませておくと、後から慌てなくて済む
- 報酬発生から支払いまでは約2ヶ月ラグがある(docs/manual.md参照)

━━━━━━━━━━━━━━━━━━━━
## 5. 初日にやること:まず初投稿10件(アプリ)
━━━━━━━━━━━━━━━━━━━━
**コレクション作成より先にこちらを済ませる。** ROOMは「コレ!」投稿が1件もないとコレクションを保存できない仕様のため、投稿→コレクション作成の順番が必須。

- [ ] 「①を実行」で今週の商品10選、「②を実行」で投稿文を用意(コピペ厳禁・口コミ分析型)
- [ ] 検索→商品ページ→「コレ!」→用意した投稿文を貼付→投稿、を10件繰り返す(棚を埋める=C条件消化)
- [ ] 自分で撮影 or 用意した写真には `#オリジナル写真` タグを付ける(Bランクの生命線、月の指定枚数を後で確認)
- [ ] 手順1で自動フォローした同ジャンルROOMerに、手動でいいね10〜20件

━━━━━━━━━━━━━━━━━━━━
## 6. コレクションを7つ作成する(アプリ・投稿後)
━━━━━━━━━━━━━━━━━━━━
手順5で投稿が1件以上ある状態になってから作成する。

**操作手順**: 画面下部の「+」アイコン →「新規コレクション」→ コレクション名を入力 → ジャンルは基本「すべてのジャンル」(複数ジャンルの商品を混在させるため) → 追加する投稿(手順5で作ったもの)を選んで保存

prompts/03_room_design.mdの構成通り、この順番・名前で作成する。

1. 🏆 楽天1位だけ集めました
2. 🥵 猛暑サバイバル(季節ごとに改名予定)
3. 🎁 夏ギフト・お中元
4. 😴 ぐっすり快眠グッズ
5. ⏰ 時短ごはんの味方
6. 🧺 スッキリ収納
7. 🛒 リピ買い定番品

投稿10件を7コレクションにどう振り分けるか迷ったら、1つの投稿を複数コレクションに重複登録してもOK(特に1「楽天1位だけ集めました」は他と重複掲載前提)。

━━━━━━━━━━━━━━━━━━━━
## 7. Day2以降のルーティンとAIチームの使い方
━━━━━━━━━━━━━━━━━━━━
- 月曜: 「①を実行」で商品10選 → 「②を実行」で投稿文一括生成 → content/に保存
- 毎日: 朝7時/昼12時/夜21時に手動投稿(15分) + いいね回り10〜20件
- 土曜: 楽天アフィリエイトのマイページ(手順3)からレポートCSVをダウンロード→data/に置く→「⑤を実行」で分析→「④を実行」で翌週カレンダー更新
- 記事(ブログ)も並行して作りたくなったら、社長として「蒐、ジャンル=◯◯でリサーチして」から記事制作チーム(prompts/team_articles/)を起動
